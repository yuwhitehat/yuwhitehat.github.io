GIT分支管理

创建分支

实际上，切换分支用switch更科学，因此，GIT提供新的git switch命令来切换分支

git merge dev 合并指定分支到当前分支

git branch -d dev

git merge –no-ff -m “merge with no–ff” dev

git stash

git stash apply
此命令恢复后，stash内容并没有删除，需要用git stash drop删除
