AI Agent是什么

AI Agent（人工智能代理）是一种能自主感知周遭环境，通过内在的智能处理进行决策，并执行相应行动以达成特定目的的智能体。其核心特征是通过与环境互动收集数据，独立规划任务路径、调用工具并执行决策，最终在无需人工干预的情况下实现预设目标。

AI Agent = LLM（核心） + Planning + Memory + Tools

Agent Skills

Skills，即技能，一开始是Claude制定的一个标准，使用的人多了，就变成了行业标准。其核心理念是让AI不再从零开始做事，而是给它一个标准化的技能工具箱，让AI扩展能力边界，尤其适用于需要重复告诉AI要怎么做一件事的场景。

一个完整的Skill，包含能力描述层、元数据层、执行逻辑层。

  • 能力描述层（对外接口）
    • 功能描述：用清晰简洁的自然语言描述功能
    • 输入/输出规范：定义触发的关键词、所需参数及返回结果的格式
  • 元数据层（管理信息）
    • 标识信息：唯一名称、版本号、开发者等
    • 分类和标签：便于检索和归类
    • 权限说明：执行所需的数据、系统访问权限清单
  • 执行逻辑层（内部实现）
    • 核心处理逻辑：实现功能的代码、脚本、工作流
    • 配置与参数：允许用户自定义的参数项
    • 错误处理机制：定义异常情况的处理逻辑
